# 🎓 Mentor Connect – Mentorship Scheduling Platform

**Mentor Connect** is a full-stack web application that enables seamless communication between mentors and students. It allows mentors to manage availability, students to book sessions, and admins to oversee platform activity. Built using **React**, **Node.js**, **Spring Boot**, and **MySQL**, it’s designed for flexibility, scalability, and ease of use.

---

## 🌟 Features

- 👩‍🏫 **Role-based Authentication**: Mentors and students have separate login functionalities.
- 📅 **Mentor Availability**: Mentors can set and manage available time slots.
- 📖 **Booking System**: Students can browse available slots and book sessions.
- 🔐 **Secure Login**: Session-based authentication using JWT and Spring Security.
- 📊 **Admin Dashboard**: Admins can manage users and sessions.
- 📱 **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices.
